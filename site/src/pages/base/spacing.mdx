# Spacing

Sleek's spacing follows a rem-based scale. Utility classes in `_space.scss` provide direct margin and padding control.

## Margin utilities

Utility classes follow the pattern `.m{side}-{step}` where side is `t/r/b/l/x/y` or blank for all sides.

```html
<div class="mt-1">margin-top: 1rem</div>
<div class="mb-2">margin-bottom: 2rem</div>
<div class="mx-1">margin-left and margin-right: 1rem</div>
<div class="m-0">margin: 0</div>
```

## Padding utilities

Same pattern with `p` prefix: `.p-{step}`, `.px-{step}`, `.pt-{step}`, etc.

```html
<div class="p-1">padding: 1rem</div>
<div class="px-2 py-1">padding: 1rem 2rem</div>
```
