# Dropdown

A custom dropdown with full keyboard navigation (arrow keys, Home/End, Enter to activate, type-to-jump). Sleek's vanilla JS auto-initializes on mount — this page's layout mounts `new Dropdown.Root(document.body)` once.

## Keyboard

- **`↓` / `↑`** — move focus to next/previous item
- **Home / End** — first / last item
- **Enter / Space** — activate focused item
- **Any letter** — jump to the next item whose label starts with that letter (type within 500ms to extend)
- **Esc** — close
